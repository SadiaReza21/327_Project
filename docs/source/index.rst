Welcome to Bazar Kori Documentation!
====================================

Bazar Kori is an e-commerce web application built with FastAPI and MySQL.

.. image:: https://img.shields.io/badge/python-3.8+-blue.svg
   :alt: Python Version

.. image:: https://img.shields.io/badge/framework-FastAPI-009688.svg
   :alt: Framework

Features
--------

* User authentication (Buyer and Admin)
* User registration and profile management
* Secure database operations
