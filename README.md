# Bazar Kori - Product Filtering System
# CSE327 Project Submission

## Project Overview
E-commerce product filtering system with FastAPI backend.

## How to Run
1. pip install -r requirements.txt
2. uvicorn filter_controller:app --reload
3. Open http://localhost:8000/filter.html
4. API Docs: http://localhost:8000/docs

## Files
- Python files: Backend with comments
- HTML/CSS/JS: Frontend with comments
- All documented during development
